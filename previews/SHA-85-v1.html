Feature preview — Downloadable Landscape Poster proposed

A new "Export" button in the toolbar renders the current filtered view (whatever search/category/ popularity state the reader has set) as a shareable static poster — title, legend, and a "as of" timestamp included — downloadable as PNG or SVG. No backend involved: pure client-side SVG serialization of the existing D3 canvas, similar to how landscape sites like CNCF's Cloud Native Landscape offer a static poster download.
