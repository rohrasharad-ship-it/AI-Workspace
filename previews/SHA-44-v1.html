Which to use where?

Two different categories

Anime.js and React Bits solve different problems. For a React/Next.js resume or Usercon site, you'd likely use React Bits (or similar) for most UI polish, and optionally Anime.js or Motion for bespoke animations.

Anime.js React Bits
What it is Animation engine (library) Component collection (copy-paste)
Framework Any (vanilla JS) React / Next.js
Integration npm i animejs npx shadcn add @react-bits/...
Best for Custom timelines, SVG morphs, physics Hero text, backgrounds, UI flair fast
Effort Higher — you compose animations Lower — pick a component, tweak props
