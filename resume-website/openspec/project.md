# Project: Resume Website

## What This Is
Sharad Rohra's portfolio site — a living product demo that shows PM craft through the site itself, including how it ships via a spec-first AI-agent loop.

## Tech Stack
- Framework: Next.js App Router
- Styling: Tailwind CSS
- Animation: GSAP
- Voice: ElevenLabs avatar
- Hosting: Vercel
- Language: TypeScript

## Non-Negotiables
- Every PR must include the Vercel preview URL
- No new dependencies without a spec proposal first
- Voice-agent FAB stays above all other fixed UI (z-index 150+)

## Out of Scope
- Blog/CMS
- Auth or user accounts
- Hand-waved marketing claims about the build process — transparency panel must show real stats
